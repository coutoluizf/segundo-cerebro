# Reminder Sound

Place a file named `reminder.mp3` in this directory to use a custom notification sound.

If no file is present, the extension will generate a simple two-tone chime using Web Audio API.

## Recommended sound properties:
- Format: MP3
- Duration: 0.5 - 2 seconds
- Sample rate: 44100 Hz or 48000 Hz
- Bitrate: 128 kbps or higher

## Free sound resources:
- [Notification Sounds](https://notificationsounds.com/) - Free notification sounds
- [Freesound](https://freesound.org/) - Creative Commons sounds
- [Zapsplat](https://www.zapsplat.com/) - Free sound effects
