Generating PNG Icons

To generate PNG icons from the SVG, you can:

1. Open this page in Chrome

2. Right-click each icon → Save Image As

3. Or use a tool like rsvg-convert or Inkscape CLI

rsvg-convert -w 128 -h 128 public/icons/raji.svg > public/icons/icon128.png
